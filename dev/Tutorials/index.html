HTBoost Tutorials

The following tutorials cover hands-on use of HTBoost. The examples provide more cases of different loss functions as well as illustrations to understand what HTBoost does and how it differs from other GBMs.

Most important user cases

  • Basic use (main options, cv, savings and loading results, variable importance and more post-estimation analysis)
  • Logistic regression (binary classification; comparison with LightGBM)
  • Time series and panels (Data Frames, time series and panels/longitudinal data, with various options for cv)
  • Categorical features (how HTBoost handles categorical features; comparison with LightGBM and CatBoost)
  • Missing data (HTBoost excels at handling missing data)
  • Speeding up HTBoost with large n (strategies to reduce computing time for large n)
  • User's controlled cross-validation( when to go beyond HTBfit for cross-validation)

Other distributions (loss functions)

  • Multiclass (multiclass classification; comparison with LightGBM)
  • Zero inflated y (loss functions for zero-inflated y; comparison with LightGBM)
  • Poisson and GammaPoisson (aka negative binomial for count data; comparison with LightGBM)
  • L2loglink and rank (a new option if min(y)≥0, whether continuous, count, or rank)
  • Robust regression (student-t and Huber; comparison with LightGBM)

Others

  • Average tau (interpreting average tau values and break-down by feature)
  • Offset (exposure) (how to add an offset, common in e.g. insurance, biology ...)
  • Beyond accurate (some suggestions when chasing maximum accuracy)
